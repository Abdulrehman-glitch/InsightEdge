\# InsightEdge



InsightEdge is an Industrial AI Twin prototype for edge-based predictive maintenance.



The system simulates industrial machine sensor readings, trains machine learning models to detect faults, exposes predictions through an API, and later visualises machine health on a dashboard.



\## Current Fault Scenarios



\- Normal operation

\- Bearing wear

\- Pressure anomaly

\- Impact event
